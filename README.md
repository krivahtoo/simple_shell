# Simple Shell
This is a simple implementation of the UNIX shell (`/bin/sh`).
This is implemented as part of the ALX SE Low-level programming & Algorithm group project.

# Build

```shell
$ make
# or
$ gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c -o hsh
```

## Run

#### Interactive
```shell
$ ./hsh
```

#### Non-interactive
```shell
$ echo "ls" | ./hsh
```
